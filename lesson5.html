সি.এস.এস এর মৌলিক বিশ্লেষণ

আবারও পুনরায় সি.এস.এস এর সাধারণ বৈশিষ্ট্য দেখি , এইবার একটু বিস্তারিত দেখবো.

  1. সর্বপ্রথমে, একটি নতুন থিম্বল পেজ তৈরি করি কাজ করার জন্য ।. আপনার এইচ.টি.এম.এল এর বডি এর মধ্যে আগে থেকেই একটি একক অনুচ্ছেদ আছে । চলুন এটি সি.এস.এস এর মাধ্যমে ডিজাইন করি।
  2. একটি <style> ট্যাগ ওপেন করি </head> এর আগে। তারপর দুইবার Enter দিন , তারপর একটি ক্লোজিং </style> দিন।. এই style এর মধ্যে আমরা আমাদের সি.এস.এস গুলো লিখবো।
  3. এইstyleএর মধ্যে এই নিচের সি.এস.এস গুলো লিখুন : 
p {
color: blue;
}

এখন আপনার একক অনুচ্ছেদ এর রং নীল হবে! বিভিন্ন অংশগুলো আরো বিস্তারিতভাবে ব্যাখ্যা :

  • এই সম্পূর্ণ অংশকে ruleset বলে (যদিও প্রায় মানুষই এটিকে শুধুrule বলে।)
  • এইখানেpহচ্ছেএকটি selector. এই selector এর মাধ্যমে এইচ.টি.এম.এল এর কোন উপাদান সিএসএস রুল দ্বারা স্টাইল করা হবে তা নির্বাচন করা হয়. এই ক্ষেত্রে, এই পেজের সকল p উপাদানগুলোকে নির্বাচন করা হচ্ছে।
  • এইখানে color: blue একটি ঘোষণাপত্র বলা হয় । আপনি একটি ঘোষণাপত্র দিচ্ছেন যে,আপনি একটি নিদিষ্ট property এইচ.টি.এম.এল এ দিতে চাচ্ছেন, সি.এস.এস value এর মাধ্যমে।.
    • color সি.এস.এস এর একটি অংশ যার মাধ্যমে আপনি একটি value দিবেন।
    • blue হচ্ছে সেই value যা আপনি সি.এস.এস দিচ্ছেন।.
  • আপনার সি.এস.এস rule গুলো অবশ্যই curly braces ( { ... } ) দ্বরা আবৃত থাকবে।

বিশেষ দ্রষ্টব্যঃ "color" এর আমেরিকার বানান. যখন ওয়েব প্রোগ্রামিং ভাষা উদ্ভাবিত হয়েছিল, চুক্তিতে ছিল যে আমরা সকলেই আমেরিকান বানান ব্যবহার করবো বিভ্রান্তি এড়াতে এবং ওয়েব প্রোগ্রামিংকে সহজ করতে।

আপনি চাইলে আপনার লেখা সি.এস.এস rule এর মধ্যে অনেকগুলি declarations যোগ করতে পারেন। আপনাকে শুধু মনে রেখে colon (:) ব্যবহার করতে হবে প্রত্যেক property এবং এর মান এর মধ্যে । (;) এর মাধ্যমে প্রতিটি declaration আলাদা করতে হবে। আপনার সিএসএস rule এর মধ্যে আরও কিছু property যুক্ত করার চেষ্টা করুন:

p {
  color: blue;
  font-size: 30px;
  background-color: yellow;
  border-radius: 10px;
  border: 2px solid green;
  padding: 10px;
}

আপনি কেমনভাবে লিখবেন এটা আপনার উপর নির্ভর করে। আমি দ্বিতীয় রকমভাবে লিখতে পছন্দ করি, এটা খুব সহজেই পড়া যায়। কি হচ্ছে এইসব

সি.এস.এস গুলো কোথায় যেতে পারে?

আমরা ইতিপূর্বে সি.এস.এস কে style উপাদান এর মাধ্যমে এইচ.টি.এম.এল এর head অংশে দিয়েছিলাম।. এটাকেInternal stylesheet বলে, কারন এটি এইচ.টি.এম এল এর মধ্যে দেওয়া হয়েছে।

পৃথক পৃথক এইচ.টি.এম.এল মধ্যে সি.এস.এস দেওয়া সম্ভব style attribute এর মাধ্যমে. উদাহরণস্বরুপ, আপনার এইচ.টি.এম.এল এর body মধ্যে নিচের উপাদানগুলো দিন:

<h1 style="text-shadow: 2px 2px 1px green;">My text rocks</h1>

একটি শিরোনাম সাধারণত সবুজ drop shadow থাকে না, আমরা এটি দিয়েছিtext-shadow property এর মাধ্যমে আমাদের style attribute এর মধ্যে। এটাকে inline CSS বলে, আলাদা ভাবে রাখার চেয়ে, এটিকে সোজা এইচটিএমএল লাইনের ভিতরে রাখা হয়। এটি করা সত্যিই খারাপ ভাবনা, আপনি যদি না পারেন আপনার এইটা করা উচিত হবে না।.

কেউ কী বলতে পারেন কেন ঠিক হবে না?

এটিকে চূড়ান্ত ভাবে একটি external stylesheet এর মাধ্যমে করা হবে, যা একটি পৃথক সিএসএস ফাইলে, পুরাপুরি আপনার এইচ.টি.এম.এল এর বাইরে থাকবে। সুতরাং আপনি example.html নামক একটি ফাইলে আপনার সব এইচ.টি.এম.এল save করতে পারেন , তারপর আপনি আপনার এইচ.টি.এম.এল ফাইল থেকে সি.এস.এস গুলো আলাদা করেstyle.css নামক ফাইল করে তার মধ্যে save করতে পারেন। (শুধু সি.এস.এস গুলো<style></style> এই tags গুলো না. তারপর আপনি আপনার সি.এস.এস ফাইলটিকে link করবেন এইচ.টি.এম.এল এর head অংশে, এটি নিচের মতো দেখাবে:

<link href="style.css" type="text/css" rel="stylesheet">

আমরা এই আধ্যয় এর বেশি আলোচনা করবো না, আমরা internal stylesheets নিয়ে Thimble এ কাজ করবো.

বিভিন্ন ধরনের নির্বাচক

আমরা এই কোর্সে কয়েকটি ভিন্ন নির্বাচক নিয়ে কাজ করবো . এ পর্যন্ত আপনি শুধু element নির্বাচক দেখেছেন। উদহারণস্বরুপ p এবং h1, যা একটি এইচ.টি.এম.এল এর মধ্যে একই ধরনের সকল উপাদান কে নির্বাচন করে। অপর দুইটি প্রধান নির্বাচক হচ্ছে class এবং id, যা এইচ.টি.এম.এল এ অবস্থিত যে কোনid or class কে নির্বাচন করে।

আইডি এবং ক্লাশ এর মধ্যে পার্থক্য হচ্ছে যে, আপনি অনেকগুলো এইচ.টি.এম.এল উপাদান যদি একই value দিতে চান তাহলে class ব্যবহার করতে হবে। কিন্তু আপনার একটি মাত্র উপাদান শুধুমাত্র একটি মাত্র value দিতে id ব্যবহার করতে হবে। ids হচ্ছে একটি বিশেষ নির্বাচক, প্রতিটি এইচ.টি.এম.এল পেজে শুধুমাত্র একটি id হতে পারে.

চলুন একটি উদহারণ দেখি. আপনার এইচটিএমএল এ অনেকগুলো paragraphs আছে, যেমন:

<p>Make something amazing with the web</p>

<p>My third paragraph</p>

<p>And another one!</p>

<p>Annnnd another...</p>

<p>My word there are loads of these!</p>

আপনি যদি কোন paragraph selector (p) ব্যবহার করেন তাহলে আপনি একসাথে সকল paragraph কে select এবং style করতে পারবেন, কিন্তু আপনি যদি শুধু একটিকে select করতে চান , অথবা তাদের মধ্যে কয়েকটি? এই ক্ষেত্রে classes এবং ids খুব সহজেই ব্যবহার করা যায়। আমাদের উপরোক্ত উদাহরণে কয়েকটি যোগ করি।

<p>Make something amazing with the web</p>

<p class="special">My third paragraph</p>

<p>And another one!</p>

<p class="special">Annnnd another...</p>

<p id="the-end">My word there are loads of these!</p>

আমরা এখন শুধু দ্বিতীয় এবং চতুর্থ paragraphs নির্বাচন করতে পারি class selector এর মাধ্যমে। class এর নামের পূর্বে (.) দিয়ে শুরু করতে হয়। উদহারণস্বরুপ:

.special {
  color: blue;
}

এবং আমরা এখন পঞ্চম নম্বর paragraph নির্বাচন করতে পারি id selector এর মাধ্যমে। id এর নামের পূর্বে (#) দিয়ে শুরু করতে হয়। উদহারণস্বরুপ

#the-end {
  color: blue;
}

বিভিন্ন ধরনের নির্বাচক রয়েছে আরো জানার জন্য: এটি দেখুন WebPlatform.org CSS selectors reference.

খেলার সময়

আমাদের পাঠ চূড়ান্ত অংশে আমরা আমাদের প্রকল্প চালু করব এবং কিছু আকর্ষণীয় সিএসএস বৈশিষ্ট্য নিয়ে কাজ করবো। কিছু content এইসব যুক্ত করার চেষ্টা করবো এবং কী পাওয়া যায় সেইটা দেখবো। আমি নিচে প্রতিটি property এর নাম এবং সাথে বর্ণনা দিয়েছি যে কোনটার কোন value দরকার। উদাহরণস্বরুপ:

  • color: আপনার উপাদানের ভিতরে text এর রং নির্ধারণ করেন। এর একটি একক সি.এস.এস colour value দিতে হবে, উদাহারণস্বরুপ color: blue;.
  • background-color: কোন উপাদানের পটভূমি রং নির্ধারন করেন। এর একটি এককসি.এস.এস colour value দিতে হবে, উদাহারণস্বরুপ background-color: red;
  • font-size: text বড় বা ছোট করতে । এই যেমন একটি নির্দিষ্ট আকার (24px) হতে পারে, একটি আপেক্ষিক মাপ (400%) অথবা অন্যান্য সি.এস.এস length values. উদাহারণস্বরুপ, font-size: 48px;
  • padding: এই কমান্ডের সাহায্যে আপনি আপনার এইচ.টি.এম.এল এর উপাদান এর চারপাশে শূণ্য স্থান এর পরিমাণ নির্ধারন করে। এই যেমন একটি নির্দিষ্ট আকার (24px) হতে পারে, একটি আপেক্ষিক আকার (30%) অথবা অন্যান্য সি.এস.এস length values. উদাহারণস্বরুপ, padding: 10px; চারটি পক্ষের একই সেট, অথবা padding: 5px 20px 10px 30px; আলাদা আলাদা করে।
  • margin: ই কমান্ডের সাহায্যে আপনি আপনার এইচ.টি.এম.এল এর উপাদান এর চারপাশে শূণ্য স্থান এর পরিমাণ নির্ধারন করে। এই যেমন একটি নির্দিষ্ট আকার (24px) হতে পারে, একটি আপেক্ষিক আকার (30%) অথবা অন্যান্যসি.এস.এস length values উদাহারণস্বরুপ, margin: 10px;চারটি পক্ষের একই সেট, অথবা margin: 5px 20px 10px 30px; আলাদা আলাদা করে।
