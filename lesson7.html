পরবর্তী:
সিএসএস লে-আউট

সিএসএস ব্যবহার করে তোমার ওয়েবপেজকে দৃষ্টিনন্দন করার অনেক রকম উপায় আছে, এবং কোর্সের এই অংশে আমি তার মধ্যে সহজগুলো নিয়ে আলোচনা করব। তোমরা হয়ত এটা জানতে খুবই আগ্রহী যে, কিভাবে তোমাদের পেজে বিভিন্ন রকম সুন্দর সুন্দর ফন্ট , ড্রপ-শ্যাডো এবং কনন্টেন্টের ব্যকগ্রাউন্ডে সুন্দর কালার ব্যবহার করবে। তাহলে চল শুরু করি।

ওয়েব ফন্ট

তুমি যদি আমার উদারণে দেওয়া টেমপ্লেটটা দেখে থাকো(উদাহরণস্বরূপ, আমার ক্রিস সম্পর্কে সবকিছু পেজটি), তাহলে তুমি কিছু সন্দর ফন্ট খেয়াল করে থাকবে , যেখানে ডিফল্ট থিম্বল পেজে সম্পন্নরূপে অন্যরকম কিছু বিরক্তিকর ফন্ট। What gives? কিভাবে আমরা আমাদের ফন্টে ভিন্নতা আনব?

শুরু করার জন্য চল সিএসএস এর ঔ অংশটা দেখি যা ফন্ট নিয়ন্ত্রণ করে। একটি নতুন থিম্বল পেজে এইচটিএমএল এর হেড অংশে style এলিমেন্ট তৈরি কর যেমনটা আমরা আগে করেছি এবং নিচের সিএসএসটি কোডটি লেখো : 

p {
  font-family: arial;
}

যখন এটা লেখা হবে, তুমি খেয়ার করবে যে ফন্ট পাল্টে গেছে। এইরকম হওয়ার কারণ, আমরা font family নিদিষ্ট করে পাল্টে দিযেছে। ফন্ট ফ্যামালী হল বিভিন্ন বিকল্প ফন্টের সংগ্রহ। উদারণ স্বরূপ, একটি ফন্ট এবং তার বোল্ড এবং ইটালিক ভারর্সন। এইা নিয়ে এখণ বেশী ভাবার দরকার নেই।

এখন pসিলেক্টরটিকে htmlসিলেক্টরে পরিবর্তন করো। তুমি খেয়াল করলে দেখবে যে খুব বেশী পরিরর্তন হয়নি। এর কারণ html এলিমেন্টের ফন্ট সেটিং উত্তরাধীকার সূত্রে তার অন্তগর্ত সব এলিমেনই পাবে। উদাহরণের জন্য HTML বডিতে কিছু এলিমেন্ট যোগ করো (হেডিং এবং প্যারাগ্রাফ ব্যবহার করলে ভাল হয়) তাহলেই তুমি দেখতে পাবে আমি আসলে কি বোঝাতে চেয়েছি। সবগুলি এলিমেন্টই html এলিমেন্টের মত একই ফন্ট ধারণ করবে।

এখন আরো একটি CSS কোড লিখো তোমার হেডিং এর জন্য। উদাহরণস্বরূপঃ

h1 {
  font-family: 'courier new';
}

নোটঃ তুমি যদি ফন্ট ফ্যামিলির নাম একাধিক অক্ষরের দিয়ে থাকো, তবে অবশ্যই নামটিকে কোটেশনের ভিতর লিখতে হবে।

সাধারণ ভাবে বলতে গেলে এই ভাবেই ওয়েবপেজে ফন্ট ব্যবহারিত হয়। তুমি একটি বেজ ফন্ট নিধারণ করে দিয়েছো এবং এটি সমস্ত ওয়েবপেজ জুড়েই প্রয়োগ হবে। তোমার যেখানে প্রয়োজন তুমি চাইলেই সেখানে এই ফন্ট পরিরর্তন করতে পারবে।

আমরা কোন ফন্টগুলো ব্যবহার করতে পারব?

পরবর্তী যে বিষয়টা বিবেচনায় রাখতে হবে তা হল তুমি শুধু মাত্র যেকোন ফন্ট ই ব্যবহার করতে পারবে না বরং তোমার পছন্দমত যেকোন জায়গাই তা ব্যবহার করতে পারবে শুধু তোমার CSS এ font-family ব্যবহার করে। ওয়েবপেজ তৈরিতে ব্যবহৃত ফন্টটি অবশ্যই ব্যবহারীর কম্পিউটারে থাকতে হবে (অন্য প্রোগ্রামগুলো যেই রকম ফন্ট ব্যবহার করে এই ফন্টগুলো সেই রকম, উদাহরণস্বরূপ, মাইক্রোসফট্ ওয়ার্ড/এই ফন্টগুলো একি রকম যেমন অন্য প্রোগ্রামগুলো ব্যবহার করে, উদাহরণস্বরূপ, মাইক্রোসফট্ ওয়ার্ড )। তোমার কম্পিউটারে কোন একটি ফন্ট থাকলে এর অর্থ এর এই নয় যে সেই ফন্ট যারা তোমার ওয়েব সাইট ভিজিট করবে তাদের কম্পিউটারেও থাকবে।

এইখানে আরো একটি সমস্যা আছে যা আমার এতক্ষণ খুজে পায়নি: @font-face এটি একটা ছোট্ট কোড যা তোমার ফন্ট ফাইলকে তোমার ওয়েব পেজের সাথে যুক্ত করার অনুমোদন দেয়, কাজেই তোমার ব্যবহৃত ফন্টটি ব্যবহারকারীর কম্পিউটারে না থাকলেও চলবে: এটা তোমার ওয়েবপেজ ফাইলের সাথে ডাউনলোড হবে। Now, implementing this yourself is a bit beyond the scope of this course,আমরা এখন আমাদের বেশীরভাগ কাজ করার জন্য একটি সাধারণ অনলাইন সার্ভিস ব্যবহার করব: Google Fonts. চলো তাহলে এটা করি!

  1. তোমার প্রজেক্ট একটি ট্যাবে ওপেন করো এবং অন্য ট্যাবে গুগুল ফন্টস।

  2. গুগুলফন্টসের পেজে তুমি ফন্টের তালিকা দেখতে পাবে এবং তুমি অপশনও ঠিক করতে পারবে।

  3. ফন্ট তালিকা দেখ এবং তা থেকে দু-তিনটি ফন্ট পছন্দ করো যা তুমি তোমার প্রজেক্টে ব্যবহার করবে। প্রত্যেকটির জন্য Add to collection বাটনে ক্লিক করো।
  4. তোমার পছন্দ করা হয়ে গেলে পেজের নিচের দিকে Use বাটনটিতে ক্লিক করো।
  5. Now you are given the code you need to get your font choices onto your page. First of all, you need to made the fonts available on your page, using the line of code seen in step 3 of the Use panel into your page. Copy this line now.
    6. <link href='http://fonts.googleapis.com/css?family=Alegreya+Sans'
      rel='stylesheet' type='text/css'>
  6. এই লাইনটি তোমার ওপেনিং <style>ট্যাগের উপরে পেষ্ট করো। এই লাইনটি তোমার সিএসএস ফাইলের উপরে থাকা জরুরি। কারণ, কোন ফন্ট ওয়েবপেজে ব্যবহার করার আগে তা ওয়েবপেজে সংযুক্ত থাকতে হবে(এইচটিএমএল কোডগুলো শুরু থেকে শেষের দিকে এইভাবে রান হয়)।
  7. Now take the lines shown in step 4 on the Google Fonts page and put them on the elements you want your fonts applied to. If you want to use a font as your base or sitewide font, you would do this:
    8. html {
  font-family: 'Alegreya Sans', sans-serif;
}
  8. ফন্ট আরো নিদিষ্টভাবে ব্যবহার করতে চাইলে, তুমি নিচের মত কোড লিখ: 
    9. h1 {
  font-family: 'Alegreya Sans', sans-serif;
}

ফন্ট সম্পর্কে এই কোর্সে আলোচনা এখানেই শেষ! তুমি ফন্ট সম্পর্কে আরো বিস্তারিত জানতে পারবে MDN font page এবং Cross browser web fonts part 1এই ওযেব সাইট দুটি থেকে।

ব্যাকগ্রাউন্ড

সংক্ষেপে আমার ফন্ট সম্পর্কে জেনেছি, এখন সময় হয়েছে সিএসএস ব্যকগ্রাউন্ড সম্পর্কে জানার। আমরা এর আগে ব্যকগ্রাউন্ড প্রোপ্রাটির খুব সাধারণ একটা অংশ: background-color দেখেছি, যেটি আমাদের কোন একটি এলিমেন্টের ব্যকগ্রাউন্ডের জন্য একটি কালার নিদিষ্ট করে দিতে দেয়। তাহলে চলো এখন আমরা ব্যবগ্রাউন্ড প্রোপ্রাটির অনান্য অপশনগুলো দেখি।

ব্যকগ্রাউন্ড ইমেজ

আমরা আগেই <img> এলিমেন্টের মাধ্যমে আমাদের পেজে ইমেজ যুক্ত করেছি । সিএসএস এর background-image প্রোপ্রাটি দিয়েও আমরা আমাদের পেজে ইমেজ যুক্ত করতে পারি। নিচের মত কোড লিখো।

p {
  background-image: url(http://www.location/of/my/image.png);
}

তুমি পেজে ইমেজ দেখতে পাবে যেমনটি আমরা চুতুর্থ সপ্তাহে দেখিছিলাম। উপরের প্রোপ্রাটিটি তোমার প্রজেক্টে ব্যবহার করার চেষ্টা কর। প্রথমে একটি ছবি নির্বাচন করো এবং এটা তোমার পেজের এক বা একাধিক এলিমেন্টের ব্যবগ্রাউন্ড হিসেবে ব্যবহার করো।

এখন, তুমি খেয়াল করলে দেখবে যে, এইচটিএমএল এ ইমেজ যেভাবে কাজ করে সেভাবে সিএসএস এ কাজ করে না! এইচটিএমএল ইমেজ নিজেদের মত করে স্পেস নেয় এব্ং তুমি একটি img এলিমেন্ট দিয়ে কেবলমাত্র একটি ইমেজ যুক্ত করতে পারবে। সিএসএস ব্যকগ্রাউন্ড ইমেজ নিজের মত স্পেস নেয় না। তার অন্য এলিমেন্টের ব্যকগ্রাউন্ড হিসেবে অবস্থান করে। এবং তারা ডিফল্ট ভাবে সমস্ত পেজ জুড়ে রিপিট হতে থাকবে ততক্ষণ পর্যন্ত যতক্ষণ না সম্পূর্ণ পেজ ভর্তি না হয়।

A very common technique is to use a seamless repeating image tile to create a nice looking background. If you search for something like "repeating background tile", you should find many suitable images to do this with. Lets try this now before we move on.

ব্যকগ্রাউন্ড রিপিট

কিছু কিছু সময় তুমি চাইবে না যে তোমার ব্যাকগ্রাউন্ড ইমেজ রিপিট হোক; তুমি হয়ত চাইবে তোমার ইমেজ আনুভূমিক বা উল্মব ভাবে রিপিট হোক কিংবা একেইবারেই রিপিট না হোক।সিএসএস background-repeatপ্রোপ্রাটি তোমাকে এটা নিয়ন্ত্রণ করার অনুমতি দেয় এবং এই একটি প্রোপ্রাটির জন্য তোমাকে চারটি ভ্যালু দেয়।

  • repeat: সাধারণত ডিফল্ট ভ্যালু নিদিষ্ট করে দেওয়া থাকে না। ইমেজ হরিজেন্টালি এবং ভার্টিকালি দুই ভাবেই রিপিট হয়।.
  • no-repeat: ইমেজ মোটেই রিপিট হয় না। শুধুমাত্র একবার দেখে দেখায়।
  • repeat-x: ইমেজ শুধুমাত্র হরিজেন্টালি রিপিট হয়, কিন্তু ভার্টিকালি রিপিট হয়না।
  • repeat-y: ইমেজ শুধুমাত্র ভার্টিকালি রিপিট হয়, হরিজেন্টালি রিপিট হয় না।

ব্যাকগ্রাউন্ড পজিশন

আরেকটি ব্যাপার, তুমি যখন কোন ইমেজকে শুধুমাত্র একবার তোমার পেজে দেখাবে তখন সেটা ডিফল্টভাবে পেজের বামদিকে উপরেরে কর্ণারে থাকবে। কিন্তু চিন্তার কোন কারণ নেই! তুমি background-position প্রোপ্রাটি ব্যবহার করে এর এটিকে ইচ্ছামত স্থানে স্থাপন করতে পারো। এটা কযেক ধরণের ভিন্ন ভিন্ন ভ্যালু গ্রহণ করে, এবং সাধারণত এটার জন্য তোমাকে দুইটি ভ্যালু দিতে হবে: একটা হরিজেন্টাল ভ্যালু এবং আরেকটা ভার্টিকাল ভ্যালু।

  1. শুরু করার জন্য তুমি কিওয়ার্ড ভ্যালু ব্যবহার করতে পারো , যেগুলো তোমার ইমেজকে নিদিষ্ট জাযগায় স্থাপন করবে; হরিজেন্টাল ভ্যালু হিসেবে left, center অথবা right, এবং ভার্টিকাল ভ্যালু হিসেবে top, center অথবা bottomব্যবহার করতে পারো। উদাহরণস্বরূপঃ, কোন ইমেজকে কোন এলিমেন্টের ডানদিকের সর্ব উপরে রাখতে চাইলে, তুমি এটা ব্যবহার করতে পারোঃ background-position: right bottom;.
  2. দ্বিতীয়ত, তুমি CSS length units ব্যবহার করতে পারো, যেটা তোমাকে আরো নিদিষ্ট ভাবে ইমেজের পজিশনিং করার ক্ষমতা দিবে। উদাহরণস্বরূপঃ, তুমি পিক্সেল ভ্যালু । এটি তোমার ইমেজকে বাম দিকের সর্ব উপর থেকে নিদিষ্ট পিক্সেল দূরে স্থাপন করতে দিবে (উদারণ, background-position: 100px 200px;), অথবা, percentages to put the image a percentage of the element's width and height across and down from the top left corner ((e.g. background-position: 50% 30%;))
  3. তুমি কিওয়ার্ড এবং সিএসএস লেনথ ইউনিট একসাথেও ব্যবহার করতে পারো, উদাহরণস্বরূপঃ background-position: 100px bottom;

অ্যাডভান্সড: শর্টহ্যান্ড

কিছুক্ষণ আগে তুমি border এবং padding এর মত প্রোপ্রাটি দেখেছ, তুমি চাইলে বিভিন্ন ধরণের ব্যকগ্রাউন্ড প্রোপ্রাটিগুলোর বদলে শর্টহ্যান্ড ভ্যালু ব্যবহার করতে পারো। এর জন্য কেবল background-* প্রোপ্রাটির বদলে background প্রোপ্রাটি লিখলেই হবে।

background-color: red;
background-image: url(http://www.clker.com/cliparts/v/R/d/R/l/c/
                      decorative-corner-design-md.png);
background-repeat: no-repeat;
background-position: right top;

তুমি শুধু এটা লিখতে পারো:

background: red url(http://www.clker.com/cliparts/v/R/d/R/l/c/
                    decorative-corner-design-md.png)
                    no-repeat right top;

ADVANCED: Drop shadows

CSS provides a mechanism to put drop shadows on your text and boxes. This is a bit more tricky to work out, but can create some fun effects.

  • text-shadow: This allows you to put a funky drop shadow on the text of your element, for example text-shadow: 2px 2px 5px black;. The four items in the value are:
    1. The horizontal offset of the shadow from the original text (how far it moves over to the right.)
    2. The vertical offset of the shadow from the original text (how far it moves toward the bottom.)
    3. The blur radius of the shadow. This means how blurry it is; a bigger number will give more blur, while 1px will give a solid shadow (no blur.)
    4. The base colour of the shadow.
  • box-shadow: This allows you to put a funky drop shadow on the your whole element, for example box-shadow: 2px 2px 5px black;. The four items in the value are:
    1. The horizontal offset of the shadow from the original box (how far it moves over to the right.)
    2. The vertical offset of the shadow from the original box (how far it moves toward the bottom.)
    3. The blur radius of the shadow. This means how blurry it is; a bigger number will give more blur, while 1px will give a solid shadow (no blur.)
    4. The base colour of the shadow.
