সিএসএস পজিশনিং

আমাদের পাঠের অষ্টম অধ্যায়ে স্বাগতম! এখানে ব্যাপারটা আপনার পছন্দের উপর নির্ভর করে। আপনি যেভাবে বর্তমানে করতেছেন সেইভাবেই করে যেতে পরেন অথবা নতুন আরো এডভান্স কিছু শিখতে পারেন। অথবা আপনি উভয়ে করতে পারেন। ব্যাপারটা আপনার উপর নির্ভর করে, আপনি দ্রুত কোন উপায়ে জিনিসটা করতে সক্ষম হচ্ছেন।

ফ্লোটস ও সিএসএস লেআউট

আপনি যখন বিভিন্ন ওয়েবসাইট দেখবেন আপনি দেখতে পাবেন, তাদের বেশিরভাগেই একটা সিঙ্গেল কলামের লেখা থাকে না পেজের উপর থেকে নিচ পর্যন্ত। বেশিরভাগেরই একাধিক কলামে লেখা সমুহ থাকে একটির সাথে আরেকটির সমন্বয় বজায় রেখে। উদাহরন স্বরুপ আপনি লক্ষ করতে পারেন মজিলা ডেভেলপার নেটওয়ার্ক, বিবিসির হোম পেজ, এবং উত্তম উদাহরন হিসেবে লক্ষ করতে পারেন, আমার‌‌ ফ্যানসইট টেমপ্লেট। তাহলে আমরা এটা কিভাবে করি?

ফ্রোটস এর পরিচয়

float, (ফ্লোটস) নামক একটি সিএসএস প্রোপার্টি আছে যেটি left (বাম) অথবা right (ডান) এর ভ্যালু গ্রহন করতে পারে। যখন আপনি এটিকে আপনার পেজের কোন এলিমেন্ট এর উপর প্রয়োগ করেন, এলিমেন্ট সমুহ এরির সোর্স বোর্ডারের নিচে বাম অথবা ডান পার্শে অবস্থান করে। আসলে এটিকে ডিজাইন করা হয়েছে ছবি সমুহকে ব্লক অথবা লেখার কাছাকাছি অবস্থান করানর জন্য, আপনি এই উপায় পত্রিকা অথবা মেগাজিনে দেখতে পারবেন। চলুন কিছু উদাহরন দেখে নেওয়া যাক।

একটি সরল সিএসএস লেআউট

ফ্লোটস যোদিও সিম্পল ফ্লোটেড ছবির জন্য উপযোগী, তবুও আপনি ফ্লোট কে আপনার ইচ্ছা মত জিনিসের উপর প্রয়োগ করতে পারবেন। ওয়েব ডেভেলপার খুব শিঘ্রই বুঝতে পারেন যে তারা ফ্লোটস দ্বারা অনেক কলাম ও লেআউট তৈরী করতে পারেন, যেটা অনেক মজাদার।

উক্ত ধারনা সম্পর্কে পরিচিত হতে, উদাহরনের বহু কলাম বিশিষ্ঠ লেআউটের টি লক্ষ করুন।

  1. থিম্বল হতে নতুন তৈরী করুনThimble page.
  2. নতুন পেজটির body (বডি)তে নিচের এইচটিএমএল টি যোগ করুনঃ:
    3. <div class="header"><h1>This is my page heading</h1></div>
<div class="main-column"><p>This is my main column.</p></div>
<div class="side-column"><p>This is my sidebar.</p></div>
<div class="footer"><p>My page © 2014.</p></div>
  3. আমরা এখন চারটি কনটেইনার এলিমেন্ট পেলাম যার ভিতর কিছু কনটেন্ট আছে। চলুন কন্টেন্টগুলির জায়গাটিকে হাইলাইট করে তাদের একটু উচ্চতা একটু বৃদ্ধি করা হোক, এখন আমরা যেটা দেখতে যাচ্ছি তা খুব সহজেই করা সম্ভব। আগেরবারের মত করে একটি style এলিমেন্ট হেড এ যোগ করুন।
  4. নিচের সিএসএসটি style এলিমেন্টে যোগ করুনঃ
    5. .header, .footer {
  background-color: green;
  height: 100px;
}

.main-column, .side-column {
  height: 200px;
}

.main-column {
  background-color: blue;
}

.side-column {
  background-color: yellow;
}
  5. চলুন এবার দুই কলাম বিশিষ্ঠ লেআউট তৈরী করা হোক। এখন প্রধান কলাম ও সাইড কলাম কে একে অপরের সাথে ফিট হওয়ার জন্য এদের প্রস্থ পরিবর্তন করা দরকার। এরা একে উপরের উপরে যেন অবস্থান না করে সেজন্য আমাদের দরকার এদের উভয়কে ফ্লোট করানো। নিচের দুটি রুল পরিবর্তন করুন যেন তা এরকম দেখায়ঃ
    6. .main-column {
  background-color: blue;
  width: 58%;
  float: left;
}

.side-column {
  background-color: yellow;
  width: 40%;
  float: right;
}
  6. এখন আপনার লেআউট টি নিচের স্ক্রিনশটের মত দেখাবে।

ফ্লোটস এর সমস্যা সমুহ সম্পর্কে আরো পরিষ্কার হওয়া

তাহলে আমাদের কাছে দুইটি কলাম বিশিষ্ঠ লেআউট আছে যাদের মাঝে একটি ফাকা জায়গা আছে, কিন্তু এটা সঠিক নয়। ফুটার হতে সবুজ কালারটি ফাকা জায়গায় চলে এসেছে, এবং ফুটারের কন্টনেন্ট গুলিকে হীন ও রংহীন মনে হচ্ছে। এর কারন হল যখন আপনি কোন এলিমেন্ট কে ফ্লোট করতে বলেন, তখন এটা পেজের নিচে ফ্লোট করে। ফুটার সহ সবকিছু একে কেন্দ্র করে ফ্লোট করবে। এইটির সমাধান হচ্ছে clear (ক্লিয়ার) প্রোপার্টি ব্যাবহার করা। যখন আপনি এটাকে কোন এলিমেন্টের উপর প্রয়োগ করেন, তখন এই প্রোপার্টি এলিমেন্টকে বলে "এখানে ভাসা বন্ধ কর"।

clear তিনটির মাঝে যে কোন একটি ভ্যালু দ্বারা গঠিত হয়ঃ left, right অথবা both। কোনটি ব্যাবহার করবেন তা নির্ভর করে আপনি কোথায় ফ্লোটিং বন্ধ করতে চান, বামের ফ্লোটিং, ডানের ফ্লোডিং অথবা ডান ও বাম উভয় পার্শে একসাথে। তবে both সবচেয়ে বেশি ব্যাবহার উপযোগী ও নিরাপদ। সমস্যাটির সমাধান করার জন্য নিচের রুল গুলি আপনার সিএসএস এর নিচে যোগ করুনঃ

.footer {
  clear: both;
}

এটি আপনার লেআউটের সমাধান হয়ত করেছে এবং তা নিচের স্ক্রিনশটের মত দেখাচ্ছে।

চ্যালেঞ্জ

এখন আপনি দুই কলাম লেআউটের বিষয় সমুহ আয়ত্ত করে ফেলেছেন, নিচের গুলি চেষ্টা করে দেখুনঃ

  • আপনার প্রজেক্টে দুই কলাম বিশিষ্ঠ লেআউট যোগ করুন।
  • আপনার দুই কলাম বিশিষ্ঠ লেআউটটিকে তিন কলাম বিশিষ্ঠ লেআউটে পরিবর্তন করুন।
  • আপনার প্রজেক্টে একটি তিন কলাম বিশিষ্ঠ লেআউট যোগ করেন।
