দশ সপ্তাহের বেসিক ওয়েব ডিজাইন কোর্স

ক্রিস মিলস, মোজিলা

সূচনা

বিভিন্ন প্রযুক্তি এবং Information and Communication Technology (ICT) এর মৌলিক বিষয় যেমন প্রোগ্রামিং সম্পর্কে জানার জন্য ওয়েব একটি ভাল যায়গা। ওয়েব সব যায়গাতে পরিচিত, আনান্য প্রোগ্রামিং এর চাইতে সহজ এবং এর মুক্ত স্বভাবের জন্য সহজেই অনেক সাহায্য পাওয়া যায়। এই কোর্সের মূখ্য উদ্দেশ্য হচ্ছে, শিশুদেরকে ওয়েব কিভাবে কাজ করে, পেছনের সেই মজার ঘটনা গুলি জানানো, HTML এর মৌলিক বিষয় (এই ভাষা ওয়েব পেজ তৈরীতে ব্যবহৃত হয়) এবং CSS এর মৌলিক বিষয় ( এই ভাষা ওয়েব পেজের প্রচ্ছদের জন্য ব্যবহৃত হয়), এর পর সময় থাকলে সম্ভবত কিছু উচ্চতর পর্যায়ের CSS এবং JavaScript (ওয়েবের প্রধান যুক্তি/প্রোগ্রামিং ভাষা)।

এই কোর্সটি ১০-১১ বছরের শিশুদের কথা চিন্তা করে বানানো হলেও ওয়েব ডিজাইনে নতুন যে কেউ অনেক কিছু শিখতে পারবে

বি:দ্র: কোর্স সম্পর্কে আপনার যেকোন মন্তব্য থাকলে বা আপনি যদি কোন পরিবর্তন দেখতে চান তাহলে গিট হাবে পুল দিতে পারেন Coding Github repo, অথবা ক্রিস কে Twitter এ মেসেজ দিতে পারেন।.

আপনার কি দরকার?

এই কোর্সের জন্য় আপনার দরকার:

  • এই লেখা টি হয় প্রিন্ট করা নতুবা অনলাইনে, যদিও অনলাইনই ভাল, কারন এখানে অনেক লিংক আছে যা কাগজে কাজ করবে না
  • ইন্টারনেট সংযোগ সহ একটি কম্পিউটার
  • একটি ওয়েব ব্রাউজার, নীচের যেকোন একটি হলে ভাল হয়:
    • ফায়ারফক্স
    • ক্রোম
    • সাফারি
    • অপেরা
    • ইন্টারনেট এক্সপ্লোরার (১০ এর উপরের সংষ্করন)
  • একটি মোজিলা ওয়েবমেকার লগইন (মূলত আপনার ইমেইল)
